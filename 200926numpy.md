### 数组的视图机制
````py

a = [1,2,3]
b = a[:]
b[0]=4
a # [1, 2, 3]


# b只是原始数据的视图，所以会修改原始数据
a = np.array([1,2,3])
b = a[:]
b[0]=4
a #array([4, 2, 3])

# 对象 b 指向 a 的对象，所以会修改原始数据
a = [1,2,3]
b = a
b[0] = 4
a #  [4, 2, 3]


````
